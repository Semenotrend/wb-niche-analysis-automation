# Авторизация WB Partners

Авторизация сделана как ручной вход через видимый Playwright-браузер.

## Команда

```bash
pnpm run login
```

## Как работает

1. Скрипт открывает `https://seller.wildberries.ru`.
2. Менеджер вручную проходит вход в WB Partners.
3. Когда личный кабинет загружен, менеджер возвращается в терминал и нажимает `Enter`.
4. Playwright сохраняет авторизованную сессию в файл:

```text
.auth/wb.json
```

## Важно

- логин, пароль, SMS-коды и 2FA не хранятся в коде;
- файл `.auth/wb.json` не должен попадать в git;
- если сессия истекла, нужно заново запустить `pnpm run login`.
