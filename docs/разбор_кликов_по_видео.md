# Разбор кликов по видео

Источник: `Как_выбрать_нишу_и_ТОПов_внише_для_сравнения.mp4`

## Что делает менеджер

1. **00:00** — находится в WB Partners на странице `Аналитика развития бизнеса`.
   - Открыта вкладка `Анализ ниш`.
   - Период выбран `Месяц`.

2. **00:04** — нажимает кнопку `Фильтры`.
   - Открывается правая панель `Фильтры`.

3. **00:06** — в панели `Фильтры` открывает поле `Категория`.
   - Поле называется `Поиск по категории`.

4. **00:08-00:09** — вводит в поле категории запрос `Бытовая техника...`.
   - В списке результатов выбирает чекбокс `Бытовая техника`.

5. **00:10-00:11** — открывает поле `Предмет`.
   - В списке предметов выбирает `Утюги`.

6. **00:13** — нажимает кнопку `Применить` в панели фильтров.
   - Фильтр применяется.
   - В таблице остается строка `Бытовая техника / Утюги`.

7. **00:14** — в строке `Бытовая техника / Утюги` нажимает правую иконку перехода/открытия.
   - Открывается карточка ниши `Утюги`.

8. **00:19** — просматривает страницу ниши `Утюги`.
   - Видны вкладки `Показатели`, `Сезонность`, `Финансовые показатели`, `Динамика по предмету`, `Поисковые запросы`.

9. **00:24** — в верхнем меню нажимает `Аналитика`.
   - Открывается выпадающее меню раздела.

10. **00:24-00:26** — в меню `Аналитика` выбирает пункт `Сравнение карточек`.
    - Пункт находится в блоке `Аналитика развития бизнеса`.
    - Открывается страница `Сравнение карточек`.

11. **00:28-00:29** — на странице `Сравнение карточек` открывает выпадающий список способа добавления карточек.
    - Изначально выбран вариант `Ввести вручную`.

12. **00:30** — выбирает вариант `Выбрать из рекомендаций по предмету`.
    - Появляется поле `Выберите предмет`.

13. **00:32-00:37** — нажимает в поле `Выберите предмет` и вводит запрос `уто`.
    - В результатах появляется вариант `Утюги`.

14. **00:37-00:38** — выбирает вариант `Утюги`.
    - Заголовок блока меняется на `Выбрать из рекомендаций по предмету «Утюги»`.

15. **00:38** — нажимает кнопку `Применить`.
    - Открывается следующий блок настроек.

16. **00:39-00:41** — в блоке `Показать топ карточек` выбирает вариант `По выручке`.
    - После выбора нажимает кнопку `Применить`.

17. **00:43** — загружается список карточек товаров слева.
    - Справа пока пусто: `Карточки для сравнения: 0 из 5`.

18. **00:47-00:48** — нажимает `Добавить →` у первой выбранной карточки:
    - `Утюг-отпариватель паровой антипригарный мощн...`
    - Артикул: `624686739`
    - Справа появляется `Карточки для сравнения: 1 из 5`.

19. **00:48-00:49** — нажимает `Добавить →` у второй карточки:
    - `Паровой утюг отпариватель с антипригарным пок...`
    - Артикул: `1022205829`

20. **00:49** — нажимает `Добавить →` у третьей карточки:
    - `Утюг паровой с антипригарным покрытием`
    - Артикул: `931189757`
    - Справа видно `Карточки для сравнения: 3 из 5`.

21. **00:49-00:50** — нажимает `Добавить →` у четвертой карточки:
    - `Утюг паровой с антипригарным покрытием мощн...`
    - Артикул: `834986705`

22. **00:51-00:52** — нажимает `Добавить →` у пятой карточки:
    - `Паровой утюг с антипригарным покрытием и отп...`
    - Артикул: `691228044`
    - Справа становится `Карточки для сравнения: 5 из 5`.

23. **00:55** — нажимает кнопку `Сравнить карточки`.
    - В записи после этого результат сравнения не показан: экран остается на странице выбора карточек.

## Итоговая последовательность кнопок

`Фильтры` → `Категория` → чекбокс `Бытовая техника` → `Предмет` → `Утюги` → `Применить` → иконка открытия строки `Бытовая техника / Утюги` → `Аналитика` → `Сравнение карточек` → выпадающий список `Ввести вручную` → `Выбрать из рекомендаций по предмету` → поле `Выберите предмет` → `Утюги` → `Применить` → `По выручке` → `Применить` → 5 раз `Добавить →` → `Сравнить карточки`.
